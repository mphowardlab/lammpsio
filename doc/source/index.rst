lammpsio documentation
======================

Tools for working with LAMMPS data and dump files. This package provides
a Python interface for reading and writing LAMMPS data files, as well as
tools for parsing LAMMPS dump files. It also allows for easy conversion
between HOOMD-Blue's GSD format and LAMMPS data files. It is designed to
be easy to use and flexible, allowing users to work with LAMMPS files
in a variety of ways that improve workflow efficiency and reproducibility.
