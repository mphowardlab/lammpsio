============
How to Cite
============

When using ``lammpsio`` in published work, please cite:

    C. L. Petix, M. Kundu, and M. P. Howard, "lammpsio: Transparent and
    reproducible handling of LAMMPS particle data in Python." *Journal of Open Source
    Software* **10**, 8962 (2025); `doi:10.21105/joss.08962 <https://doi.org/10.21105/joss.08962>`_.
